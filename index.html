Noowit

Wordpress plugin for posting directly from your blog to your NooWit Magazine

View project onGitHub

About

This is an independent developed Wordpress plugin for posting directly from your blog directly to your Noowit Magazine. The plugin isn't available in the default Wordpress repository, so you have to manually install it. For more information on manually installing plugins you can refer to Wordpress Documentation .

This plugin depends highly on curl php library, so you have to make sure that it is installed in your server for it to function properly.

Configuration

After installation you can find Noowit configuration options under Settings Menu in Administrator interface.

There, you have to specify you authentication key which you can obtain from "My MAG Settings" section under Settings in Noowit Web Application. The key is needed in order to establish correct communication to Noowit server and authenticate against it so that your posts are properly published under your Noowit Magazine.

Use

There are two publishing options available.

Automatically publishing a post to your Noowit Magazine whenever you hit the Publish button, or manually publishing your post. In the latter case, an extra Noowit Publish button appears above the Publish button to support this functionality.
